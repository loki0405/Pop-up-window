# Pop-up-window
JS原生组件，页面弹窗
api说明
构造函数 Dialog
参数init:
id:0,                      //id必须传
width:400,                 //弹窗的宽度
height:200,                //弹窗的高度  
pos:'center',              //弹窗的位置
title:'浮生如梦',           //弹窗的标题
content:'当浮于大白',       //弹窗的内容
suerButton:false,          //弹窗是否有确认键
sureButtonText:'确定',     //'确认键'的文字内容
mask:true  	               //弹窗是否有遮罩层
